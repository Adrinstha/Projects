# Nepal Construction Company Website

A modern, responsive construction company landing page built using HTML5 and CSS3. The project demonstrates responsive web design principles, semantic HTML structure, SEO fundamentals, and mobile-first development practices.

## Features

* Responsive navigation bar
* Mobile-friendly layout
* Hero section with call-to-action
* Services section with responsive card grid
* Why Choose Us section
* Contact CTA section
* SEO-friendly HTML structure
* Semantic HTML5 elements
* Smooth scrolling navigation
* Modern CSS Grid and Flexbox layouts

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries

## SEO Features

* Optimized page title
* Meta description
* Single H1 structure
* Proper heading hierarchy
* Descriptive image alt text
* Semantic HTML tags
* Mobile responsive design

## Responsive Breakpoints

| Device  | Width   |
| ------- | ------- |
| Mobile  | < 576px |
| Tablet  | < 992px |
| Desktop | > 992px |
